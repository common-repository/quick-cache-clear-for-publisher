== Description ==
Allows anyone with publish permissions to clear the cache in Quick Cache

Note: This currently only works for WordPress Multisite. I've put in a request for a new hook in Quick Cache to correct the problem with WordPress.

== Installation ==

1. Copy the plugin files to <code>wp-content/plugins/</code>

2. Network Activate plugin from Network Plugins page or activate as regular plugin

3. Enjoy!
